# JAIPilot - High-Coverage Java Unit Tests

Automate Java unit tests for pull requests and raise coverage with PR-ready updates.

## Why teams use it

- Raises coverage on changed Java code.
- Generates PR-ready test updates.
- Validates tests compile and run.

## How it works

1. Analyze changed Java code.
2. Generate or update tests and validate execution.
3. Open a pull request with test changes.

Install JAIPilot for GitHub: [https://github.com/apps/jaipilot/installations/new](https://github.com/apps/jaipilot/installations/new)
